## Vector Dimensions

- **128-256** : For small datasets
- **384-512** : For text embeddings (sentence-transformers)
- **768-1536** : For advanced models (BERT, etc.)

## Distance

- **COSINE** : Recommended for texts, normalized embeddings
- **EUCLID** : For numerical data
- **DOT** : For non-normalized vectors
