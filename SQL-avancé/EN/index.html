    Advanced SQL/PostgreSQL Training - Query Optimization¶

    📚 Overview¶

    This training covers advanced SQL/PostgreSQL query optimization techniques, with a particular focus on using Dalibo for performance analysis and optimization.

    🎯 Learning Objectives¶

    • Understand PostgreSQL query execution mechanisms
    • Master advanced optimization techniques
    • Use Dalibo to analyze and optimize performance
    • Interpret key performance indicators
    • Apply best practices in real-world scenarios

    📖 Training Structure¶

    1. Optimization Fundamentals¶

    • PostgreSQL architecture and query planner
    • Index types and their usage
    • Statistics and ANALYZE

    2. Execution Plan Analysis¶

    • EXPLAIN and EXPLAIN ANALYZE
    • Interpretation of operations (Seq Scan, Index Scan, etc.)
    • Costs and execution time

    3. Dalibo - Analysis Tool¶

    • Installation and configuration
    • Query analysis with pg_stat_statements
    • Performance reports
    • Automatic recommendations

    4. Performance Indicators¶

    • Key metrics to monitor
    • Interpretation of Dalibo indicators
    • Alert thresholds and best practices

    5. Optimization Techniques¶

    • Join optimization
    • Aggregation optimization
    • Subquery optimization
    • Partitioning and parallelism

    6. Practical Cases¶

    • Real-world optimization scenarios
    • Before/After with metrics
    • Common problem resolution

    7. Exercises¶

    • Guided exercises
    • Problems to solve
    • Commented solutions

    🚀 Quick Start¶

    1. Prerequisites
    2. PostgreSQL 12+ installed
    3. Access to a test database

    4. pg_stat_statements extension enabled

    5. Dalibo Configuration 

      -- Enable pg_stat_statements
CREATE EXTENSION IF NOT EXISTS pg_stat_statements;

    6. Follow the Training

    7. Start with Module 1 (Fundamentals)
    8. Follow the module order for logical progression
    9. Practice with Module 7 exercises

    📊 Recommended Tools¶

    • Dalibo: PostgreSQL performance analysis
    • pgAdmin: Graphical interface for PostgreSQL
    • psql: Command-line client
    • EXPLAIN Visualizer: Execution plan visualization

    📝 Conventions¶

    • SQL examples are tested on PostgreSQL 14+
    • Metrics are based on typical production environments
    • Execution times may vary depending on configuration

    🤝 Contribution¶

    This training is designed to be scalable. Feel free to suggest improvements or additional use cases.
