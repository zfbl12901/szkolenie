## üöÄ Quick Start

### Step 1: Create a Free Azure Account

1. Go to: https://azure.microsoft.com/free/
2. Click "Start free"
3. Sign in with Microsoft account
4. Verify your identity by phone
5. Enter credit card information (not charged)

**Important:** Azure gives you $200 credit for 30 days, then permanent free services.

### Step 2: Explore Free Services

**Free services useful for Data Analyst:**

- **Azure SQL Database** : Free up to 32 GB (12 months)
- **Azure Storage** : 5 GB (12 months)
- **Azure Data Factory** : Free up to 5 pipelines
- **Azure Functions** : 1 million executions/month (always free)
- **Azure Cosmos DB** : 400 RU/s (always free)

### Step 3: Follow the Training

1. Start with module 1 (Getting Started)
2. Follow module order
3. Use Microsoft Learn for labs
4. Practice with module 7 projects
5. Monitor your usage in Azure Cost Management

## ‚ö†Ô∏è Cost Management

### Tips to Stay Free

1. **Monitor Billing**
   - Enable cost alerts
   - Regularly check Azure Cost Management
   - Recommended limit: $5 alert

2. **Use Free Services**
   - Prioritize always-free services
   - Use free credits wisely
   - Stop unused resources

3. **Delete Unused Resources**
   - Stop virtual machines
   - Delete empty storage accounts
   - Regular cleanup

4. **Use Free Regions**
   - Some regions offer more free services
   - Check availability by region
